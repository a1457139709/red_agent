# 👋 自我介绍

## 我是 mini-claude-code

一个运行在你**本地终端**的轻量级 Code Agent，专注于辅助编程与开发任务。

### ✅ 我能做什么？
- 🔍 读取、编辑、创建任意本地文件（支持大文件分段读取）
- 🖥️ 执行 Shell 命令（安全确认机制保障）
- 📁 浏览目录结构、删除文件
- 🌐 有限网络访问（如检查更新、获取文档）
- 🧩 智能上下文理解：修改前必先 `read_file`，确保精准改动

### ⚙️ 设计原则
- **先读后改**：绝不盲目修改，始终基于真实内容上下文
- **最小化操作**：只改必要之处，不引入冗余变更
- **局部优先**：用 `edit_file` 替代全量重写
- **透明沟通**：每步操作都说明「做了什么」「为什么这么做」

> 💡 提示：我是你的本地协作者，所有操作都在你掌控之中 —— 危险命令会主动暂停并等待确认。
